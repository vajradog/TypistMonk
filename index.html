TypistMonk provides a simple, easy and quick way to edit a sample letter or write your own on the browser and print it with ease, without the need to open a text editor or login to an online service etc. Basically a 'point-and-shoot' text editor.

I started and built the site with the amazing Ruby on Rails framework. Eventhough the current site does not use much of what Rails offer, however as the Tibetan saying goes,"build the dam before the flood comes", the decision to use rails was made to facilitate any dynamic feature we may need in the future.

The text editor is based on David Ferreira's clone of the beautiful inline editor toolbar medium editor and was written in vanilla Javascript.

The front-end framework is Twitter's bootstrap. This website lives in Github, runs from Heroku and was made with love by Thupten N. Chakrishar.
